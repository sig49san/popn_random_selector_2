### 要件フェーズ (2026-02-06 12:05)

#### アプリケーションの目的

- ポップンミュージックの楽曲データからランダムに楽曲を選出する。
- 選出された楽曲は、ユーザーが設定した件数（1～20曲）をリスト形式で表示する。

#### UI/UX

- レイアウトは `./Reference/popn_random_selector/` に存在する既存のNext.jsアプリケーションを参考にすること。
- ユーザーが選出楽曲数を設定できるUI要素（例：スライダー、入力フィールドなど）を設ける。

#### データソース

- 楽曲データは `popn_music_list.csv` ファイル (または同等のCSVファイル) を利用する。
- CSVファイルはアプリケーションで利用できるようJSON形式に変換して読み込む仕組みを導入する。

#### 技術スタック

- ビルドツール: Vite
- UIライブラリ: React
- プログラミング言語: TypeScript
- UIコンポーネントライブラリ: Material UI
- テストフレームワーク: React Testing Library, Vitest

#### デプロイメント

- GitHub Pagesでの公開を想定したプロジェクト設定を行う。

#### 開発環境

- 開発者が容易にプロジェクトをセットアップ、開発、実行、テストできる環境を構築する。
- 必要な依存関係を明確にし、インストール手順を提供すること。
